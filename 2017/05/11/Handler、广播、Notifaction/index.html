Runnable线程的用法    
1、先创建一个Thread实例，然后实现Run方法、通过Run方法的Handler启动    我们要启动的Runnable线程对象。
2、创建Runnable对象实现Run方法，写具体的操作
然后用Handler的方法多次调用runnable线程对象，多次启动。

    第一个参数是，要启动的runnable线程对象。
    第二个参数是每隔多少毫秒启动一次
handler.postDelayed(runnable, 1000);

    从handler中移除runnable对象
handler.removeCallbacks(runnable);

    启动Runnable的方法
        new Thread(){
                public void run() {
                handler.postDelayed(runnable, 0);
                };
            }.start();
        }
    Runnable的执行方法  （实现倒计时）
    int i = 60;
    public void run() {
            if(i>0){
                i--;
                textview.setText(i+"");
                handler.postDelayed(runnable, 1000);
            }else {
                handler.removeCallbacks(runnable);
            }
        }
    };

Broadcast Receiver（广播接收器）
静态注册和动态注册

1、静态注册可以开机自启动，但是无法取消注册不够灵活
    须要在AndroidManifest当中写一组<Receiver><Receiver/>标签
    <Receiver>当中name要写好继承Broadcast Receiver的包名和类名
    然后在Receiver当中写<intent-filter>标签，标签中写<action>
    <action>当中的name须要对应要接收的广播



2、动态注册可以在方法中注册。

Notification

先创建一个通知管理器，
然后再创建一个通知
然后需要创建两个Intent，一个Intent，一个PendingIntent
然后设置通知里面具体的内容，要是想点击跳转就把pendingIntent加上
可以设置声音，设置自动取消…………
添加到通知管理器当中

//得到通知栏
NotificationManager manager=(NotificationManager) getSystemService(NOTIFICATION_SERVICE);

//创建Notification设置图片，内容，时间
Notification notification=new Notification(R.drawable.ic_launcher, "MC刘哲", System.currentTimeMillis());

Intent it=new Intent(MainActivity.this,NumActivity.class);
//requestCode 如果不相同，就会重新创建一个pendingIntent
//FLAG_ONE_SHOT 如果PendingIntent已经存在，就不会去更新PendingIntent里面封装的Intent
PendingIntent pendingIntent=PendingIntent.getActivity(MainActivity.this, 0, it, PendingIntent.FLAG_ONE_SHOT);
//设置notification详细内容
notification.setLatestEventInfo(MainActivity.this, "MC刘哲", "如花我爱你，嫁给我好吗", pendingIntent);
//设置声音
notification.flags=Notification.FLAG_AUTO_CANCEL;
//设置自动取消
notification.defaults=Notification.DEFAULT_SOUND;
//将notification弹出
manager.notify(0, notification);
